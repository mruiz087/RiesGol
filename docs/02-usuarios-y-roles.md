# Roles

## Member

Puede:

- Apostar.
- Elegir favoritos.
- Ver clasificaciones.
- Ver estadísticas.

## Admin

Puede:

- Crear porras.
- Modificar la configuración.
- Cambiar premios especiales.
- Gestionar usuarios.
- Promocionar administradores.
- Retirar administradores.

## Regla importante

Una porra debe tener siempre al menos un administrador.

No se permitirá:

- Eliminar al último administrador.
- Degradar al último administrador.
