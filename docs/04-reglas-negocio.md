# Alias

Los alias son únicos globalmente.

No se permiten duplicados.

Las comparaciones ignorarán mayúsculas y minúsculas.

Ejemplo:

Marcos
MARCOS
marcos

Representan el mismo alias.

# Incorporación de participantes

No se permiten nuevas incorporaciones una vez iniciado el torneo.

# Empates

Los empates son reales.

Si dos usuarios obtienen la misma puntuación:

- Comparten posición.
- Ambos son considerados premiados.

# Históricos

Las porras nunca se eliminan automáticamente.

Pueden permanecer:

- Activas
- Finalizadas
- Archivadas
