# Selección obligatoria

Cada usuario debe elegir:

- 1 equipo del bombo 1
- 1 equipo del bombo 2
- 1 equipo del bombo 3
- 1 equipo del bombo 4

# Bloqueo

Los favoritos pueden modificarse hasta el inicio del primer partido del torneo.

A partir de ese momento:

Solo lectura.

# Cálculo de gol

Factor gol =

Puntos FIFA favorito principal
/
Puntos FIFA equipo seleccionado

# Ejemplo

España = 2000

Colombia = 1700

Factor =

2000 / 1700 = 1.18

Cada gol de Colombia:

1.18 puntos

# Multiplicadores

Los goles utilizan el mismo multiplicador que las apuestas.

# Clasificación Pichichi

Mostrar únicamente:

- Posición
- Usuario
- Puntos

# Modelo

La selección de favoritos no almacena puntos.

Los puntos se calculan dinámicamente.
