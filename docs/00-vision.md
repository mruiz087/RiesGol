# ReisGol

ReisGol es una plataforma web de porras privadas para torneos internacionales de fútbol.

Inicialmente soportará:

- Mundial
- Eurocopa

Objetivos:

- Crear porras privadas.
- Invitar usuarios mediante códigos.
- Gestionar apuestas 1-X-2.
- Mantener clasificaciones dinámicas.
- Implementar competición Pichichi basada en equipos favoritos.
- Mantener histórico de torneos y porras finalizadas.

La plataforma debe soportar múltiples torneos, múltiples porras y múltiples administradores desde la primera versión.
