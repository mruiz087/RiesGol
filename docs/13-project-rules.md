# Reglas para agentes

Utilizar TypeScript.

No utilizar any.

Crear componentes reutilizables.

Separar lógica y presentación.

Validar permisos en backend.

Todas las consultas deben incluir:

group_id

tournament_id

cuando corresponda.

No utilizar IDs hardcodeados.

No duplicar datos calculables.

Mantener soporte:

- multiporra
- multitorneo
- múltiples administradores

Priorizar mantenibilidad sobre optimización prematura.
