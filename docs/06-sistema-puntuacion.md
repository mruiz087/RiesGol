# Regla principal

Cada usuario obtiene tantos puntos como participantes hayan fallado el partido.

# Ejemplo

10 usuarios

2 aciertan
8 fallan

Puntos obtenidos:

8

# Multiplicadores

Fase grupos      x1

Dieciseisavos    x2

Octavos          x3

Cuartos          x4

Semifinales      x5

Final            x6

# Fórmula

Puntos partido =

Usuarios que fallan
x
Multiplicador fase
